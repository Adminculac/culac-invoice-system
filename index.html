ℹ️
ตั้งค่า Google Apps Script URL เพื่อใช้ Google Sheets เป็น Server
ขั้นตอน:
  1. สร้าง Google Spreadsheet ใหม่
  2. Extensions → Apps Script
  3. Copy Code.gs ไปวาง
  4. Deploy → New deployment → Web app
  5. Execute as: Me / Who has access: Anyone
  6. Copy URL มาใส่ด้านล่าง
🔢 ตั้งค่าเลขที่ใบแจ้งหนี้
ℹ️
รูปแบบ: XXYYMM (XX=ลำดับ/YY=เดือน/MM=ปี พ.ศ.)
ตัวอย่าง: 010569 = ใบแรก เดือน พ.ค. ปี 2569
หมายเหตุ: ระบบจะรีเซ็ตเป็น 01 ทุกต้นเดือนอัตโนมัติ
กรณีเริ่มใช้งานกลางเดือน หรือต้องการเปลี่ยนเลข
ใส่เลขใบล่าสุดที่ออกไปแล้ว (6 หลัก) ระบบจะออกใบถัดไปต่อให้ เช่น 030669 → ใบถัดไป 040669
พิมพ์เลขที่ต้องการให้เป็น "ใบถัดไป" ได้เลย (6 หลัก) ระบบจะใช้เลขนี้ทันที ไม่ +1
